Provjera Statusa Predmeta

Ova skripta provjerava status predmeta u sudkom registru, za zadani ID u slucaju da se promijeni status, salje Telegram poruku sa obavjestenjem.

Kako podesiti:

- Napravi telegram bota
- Podesi .env file sa podacima (ID predmeta, Telegram bot token i ID chata, URL ostaviti is examplea za registar 2025)
- npm install
- Podesiti cron job da ga runa u odredjeno vrijeme (9 ujutro je recommended)
